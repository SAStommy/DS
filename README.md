# 筆記區
## set()
他可以把裡面的惟一字元抽出來形成新的集合
### e.g
a = "abcdaee"\
set(a)={a,b,c,d,e}
## zip()
他可以把多個集合的對應element多個新的小集合
### e.g.1
o = [1,2,3]\
p = {a,b,c}\
q = {x,y,z}\
zip(o,p,q)=(1,a,x),(2,b,y),(3,c,z)
### e.g.2
o = [1,2,3]\
p = {a,b}\
zip(o.p)=(1,a),(2,b)
